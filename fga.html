Fine Grained Auditing (FGA)

Главная Unified Auditing FGA Традиционный аудит Управление Архитектура Безопасность Инструменты

Что такое FGA

FGA позволяет вести аудит на уровне отдельных строк и столбцов, фиксируя события только при срабатывании условий.

Пример включения FGA


BEGIN
  DBMS_FGA.ADD_POLICY (
    object_schema => 'HR',
    object_name   => 'EMPLOYEES',
    policy_name   => 'salary_read_control',
    audit_condition => 'salary > 20000',
    audit_column => 'SALARY'
  );
END;
/
