Fine-Grained Auditing (FGA)

FGA позволяет фиксировать только те запросы, которые соответствуют определённым условиям. Критически важно для контроля доступа к конфиденциальным данным.

Пример политики FGA


BEGIN
    DBMS_FGA.add_policy(
        object_schema   => 'HR',
        object_name     => 'EMPLOYEES',
        policy_name     => 'salary_select',
        audit_column    => 'SALARY'
    );
END;
/

Просмотр логов FGA


SELECT event_timestamp, db_user, object_name, sql_text
FROM dba_fga_audit_trail
ORDER BY event_timestamp DESC;
