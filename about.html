О проекте Oracle Audit

Этот учебный веб-проект посвящён теме аудита действий пользователей в Oracle Database.

Цели проекта

  • Предоставить понятное объяснение механизмов аудита
  • Показать разницу между типами аудита
  • Дать реальные примеры SQL-запросов
  • Показать современные практики безопасности

Технологии

  • HTML5 — структура страниц
  • CSS3 — адаптивное оформление
  • JavaScript — интерактивность
  • Oracle SQL — примеры запросов

Контакты
