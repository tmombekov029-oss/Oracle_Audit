Добро пожаловать в систему аудита Oracle

Этот учебный проект посвящён механизмам аудита в Oracle Database. Здесь вы найдёте информацию о различных типах аудита, практические примеры и рекомендации по безопасности.

Основные разделы

  • Unified Auditing - современный механизм аудита
  • Fine-Grained Auditing - детальный аудит на уровне строк
  • Classic Auditing - классический подход к аудиту
  • Инструменты - средства анализа и мониторинга
  • Примеры SQL - готовые запросы для работы
