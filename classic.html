Classic Auditing

Классический аудит — традиционный метод регистрации действий пользователей, существующий в Oracle с ранних версий.

Включение классического аудита


ALTER SYSTEM SET audit_trail='DB,EXTENDED' SCOPE=SPFILE;
SHUTDOWN IMMEDIATE;
STARTUP;

Примеры команд аудита


AUDIT SESSION;
AUDIT SELECT ON hr.employees;
AUDIT INSERT, UPDATE, DELETE ON hr.employees;

Просмотр результатов


SELECT username, action_name, obj_name, timestamp
FROM dba_audit_trail
ORDER BY timestamp DESC;
