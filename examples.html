Примеры SQL для аудита Oracle

Реальные примеры SQL-запросов для анализа логов аудита и отслеживания действий пользователей.

Просмотр всех записей Unified Auditing


SELECT event_timestamp, username, action_name, sql_text
FROM unified_audit_trail
ORDER BY event_timestamp DESC;

Поиск ошибок доступа


SELECT event_timestamp, username, return_code
FROM unified_audit_trail
WHERE return_code != 0
ORDER BY event_timestamp DESC;

Аудит DDL операций


SELECT event_timestamp, username, action_name, object_name
FROM unified_audit_trail
WHERE action_name IN ('ALTER TABLE', 'DROP TABLE', 'CREATE USER')
ORDER BY event_timestamp DESC;
