Unified Auditing в Oracle Database

Unified Auditing — это современная архитектура аудита, введённая в Oracle Database 12c. Она объединяет все типы аудита в единую централизованную подсистему.

Основные преимущества

  • Высокая производительность
  • Централизованность настроек
  • Улучшенная защита логов
  • Интеграция со всеми механизмами Oracle

Пример создания политики


CREATE AUDIT POLICY logins
    ACTIONS LOGON;

AUDIT POLICY logins;

Просмотр записей аудита


SELECT event_timestamp, username, action_name, sql_text
FROM unified_audit_trail
ORDER BY event_timestamp DESC;
