Управление аудитом и журналами

Настройка хранения логов, очистка устаревших записей и интеграция с SIEM.

Компоненты управления

  • Архивирование журналов аудита
  • Очистка устаревших записей
  • Интеграция с SIEM-системами
  • Генерация отчётов
  • Мониторинг производительности

Примеры SQL для управления


-- Просмотр политик аудита
SELECT policy_name, enabled, description
FROM dba_audit_policies;

-- Очистка старых записей аудита
DELETE FROM dba_audit_trail
WHERE timestamp < SYSDATE - 90;

-- Настройка архивирования
BEGIN
    DBMS_AUDIT_MGMT.SET_LAST_ARCHIVE_TIMESTAMP(
        audit_trail_type => DBMS_AUDIT_MGMT.AUDIT_TRAIL_UNIFIED,
        last_archive_time => SYSDATE - 30
    );
END;
/

Автоматизация очистки


-- Создание задачи очистки
BEGIN
    DBMS_AUDIT_MGMT.CREATE_PURGE_JOB(
        audit_trail_type => DBMS_AUDIT_MGMT.AUDIT_TRAIL_UNIFIED,
        audit_trail_purge_interval => 24,
        audit_trail_purge_name => 'DAILY_PURGE_JOB'
    );
END;
/

Мониторинг использования


-- Статистика по аудиту
SELECT action_name, COUNT(*) as event_count
FROM unified_audit_trail
WHERE event_timestamp > SYSDATE - 7
GROUP BY action_name
ORDER BY event_count DESC;

-- Размер журналов аудита
SELECT segment_name, bytes/1024/1024 as size_mb
FROM dba_segments
WHERE segment_name LIKE '%AUDIT%';
