Управление аудитом

Настройка хранения логов, очистка устаревших записей и интеграция с SIEM.

Примеры SQL


-- Просмотр политик
SELECT policy_name, enabled, description
FROM dba_audit_policies;

-- Очистка старых записей
DELETE FROM dba_audit_trail
WHERE timestamp < SYSDATE - 90;
